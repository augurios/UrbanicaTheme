A) Installation - top

  1. Login to your Tumblr account
  2. Open up your Tumblr page. It should be 'yourusername'.tumblr.com
  3. On the top right corner click on the Edit Theme button
  4. Now click the Edit HTML button
  5. Copy the code from index.html file to Tumblr and click the Save button
  6. You can use any text editor to copy the index.html code, for example Notepad on Windows.
  7. Click on the 'Back' Arrow button
  8. Your Theme is ready to use!

B) Custom Settings - top

Inside the 'Edit Theme' panel, bellow the default APPEARANCE OPTIONS you can find the custom THEME OPTIONS these are:

Image options

You can upload custom images for the main Background and Header background image.

Colors

You can customize the highlight color for each post.

Font

You can change the font for the page body.

Toggles

You can turn on and off several features of the theme, these include: Show Description on Index, Show search on sidebar, Snow mode(this change the feel from dark to light version), show following, show tweet button, show like button.

Text fields

In this options you can customize some labels as well as enter your respective username for the diverse social services available on the theme.
